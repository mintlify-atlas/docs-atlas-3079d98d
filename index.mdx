## What is RideShare Live?

RideShare Live is a production-ready ride-sharing platform built with **Spring Boot 3** on the backend and **React 19** on the frontend. It supports two user roles — **Riders** and **Drivers** — with separate dashboards and real-time coordination via WebSocket.

<Info>
  RideShare Live is designed for self-hosting. You bring your own PostgreSQL database, Redis instance, Stripe account, and email credentials.
</Info>

## Get started

<Steps>
  <Step title="Set up prerequisites">
    Install Java 17, Maven, PostgreSQL 14+, Redis, and Node.js 18+. See the [quickstart guide](/quickstart) for platform-specific instructions.
  </Step>
  <Step title="Configure environment">
    Copy `.env.example` to `.env` and fill in your database credentials, JWT secret, and third-party API keys. See [environment variables](/configuration/environment-variables).
  </Step>
  <Step title="Start the backend">
    Run `mvn spring-boot:run` from `ridesharelive-backend-main/`. The API will be available at `http://localhost:8080`.
  </Step>
  <Step title="Start the frontend">
    Run `npm install && npm run dev` from `ridesharelive-frontend-main/`. The app will be available at `http://localhost:5173`.
  </Step>
</Steps>
